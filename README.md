<h2> Sobre o projeto</h2>

<p>
  O <strong>Personal Portfolio Practice</strong> é uma aplicação web desenvolvida como atividade prática do bootcamp da
  <strong>Generation Brasil</strong>. O projeto simula uma página de portfólio pessoal, permitindo apresentar informações,
  projetos e contato em uma interface organizada, responsiva e interativa.
</p>

<p>
  Além da estrutura visual, a aplicação consome dados da <strong>GitHub API</strong> para exibir informações dinâmicas
  do perfil e dos repositórios, tornando o conteúdo mais automatizado e atualizado.
</p>

<p>
  <strong>Importante:</strong> este repositório <strong>não representa o portfólio oficial da desenvolvedora</strong>,
  mas sim um projeto de prática criado com fins educacionais para exercitar conceitos fundamentais de desenvolvimento front-end.
</p>

<hr />

<h2> Objetivos de aprendizagem</h2>

<ul>
  <li>Praticar a construção de páginas com <strong>HTML semântico</strong></li>
  <li>Desenvolver layouts modernos e responsivos com <strong>CSS</strong></li>
  <li>Aplicar interatividade com <strong>JavaScript</strong></li>
  <li>Consumir dados externos utilizando a <strong>GitHub API</strong></li>
  <li>Trabalhar organização visual, navegação e experiência do usuário</li>
  <li>Integrar recursos como carrossel e validação de formulário</li>
</ul>

<hr />

<h2> Funcionalidades</h2>

<ul>
  <li>Estrutura de páginas desenvolvida com <strong>HTML semântico</strong></li>
  <li>Estilização moderna com <strong>CSS</strong>, utilizando variáveis, animações e responsividade</li>
  <li>Integração com a <strong>GitHub API</strong> para exibição dinâmica do perfil e dos repositórios</li>
  <li>Exibição dos projetos em <strong>carrossel interativo</strong> com <strong>Swiper.js</strong></li>
  <li><strong>Formulário de contato</strong> com validação no front-end</li>
  <li>Página de <strong>confirmação de envio</strong> do formulário</li>
  <li>Navegação fluida com menu fixo e rolagem suave</li>
  <li>Interface organizada, pensada para boa experiência em desktop e mobile</li>
</ul>
